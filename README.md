msbuildspecs
============
为多种工作准备的 msvc配置文件

目前包含
--------

* msvc2010 默认配置
* Daffodil.ENU 包括 vc60,vc70,vc71,vc80 配置
* autocad 为AutoCAD/ObjectARX开发准备的配置文件

开发/测试环境
-------------

* windows 7 64bit
* visual studio 2010 
* msbuild v4.0
